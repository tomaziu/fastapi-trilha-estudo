Repositório com meu progresso de aprendizado em FastAPI, do básico ao avançado. Aqui você encontra exercícios práticos, anotações, projetos e desafios organizados por tópico.

> Este projeto é baseado no meu próprio estudo e evolução com o framework. Cada exercício foi desenvolvido durante minha jornada de aprendizado, com o objetivo de consolidar os conceitos de forma prática.

---

## Pré-requisitos

- Python 3.10+ instalado → [python.org/downloads](https://www.python.org/downloads/)
- Noções básicas de terminal/linha de comando
- Editor de código (recomendo VS Code)

---

## Como usar este repositório

As pastas dentro de `meu_estudo_fastapi/` estão numeradas e devem ser seguidas **em ordem**.  
Cada pasta contém:

- Anotações e explicações do tópico
- Exercícios práticos para fixar o conteúdo
- Exemplos de código comentados

Siga do `01_fundamentos/` até o `12_projeto_final/` para uma trilha completa do básico ao avançado.

---

## Estrutura

```
meu_estudo_fastapi/
├── sessao_fastapi.md              # Controle de progresso e histórico
├── 01_fundamentos/                # Instalação, primeira rota, Swagger
├── 02_rotas_http/                 # GET, POST, PUT, DELETE, status codes
├── 03_parametros/                 # Path, Query, Body, Headers
├── 04_pydantic/                   # Validação de dados, models, schemas
├── 05_organizacao_projeto/        # Routers, files, dependency injection
├── 06_sqlalchemy/                 # ORM, banco de dados, CRUD
├── 07_auth_jwt/                   # Autenticação, tokens, login
├── 08_middlewares_cors/           # Middlewares, configuração de CORS
├── 09_tratamento_erros/           # HTTPException, handlers customizados
├── 10_testes/                     # Pytest, TestClient, cobertura
├── 11_boas_praticas/              # Estrutura, Docker, deploy
└── 12_projeto_final/              # Projeto completo aplicando tudo
```

---

## Como instalar

```bash
# Clonar o repositório
git clone https://github.com/tomaziu/fastapi-trilha-estudo.git
cd fastapi-trilha-estudo

# Criar ambiente virtual
python -m venv venv

# Ativar (Windows)
venv\Scripts\Activate.ps1

# Ativar (Linux/Mac)
source venv/bin/activate

# Instalar dependências
pip install -r requirements.txt
```
